# v0.2 Feature Batch Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Harden and extend renovate-server: webhook group allowlist, renovate-marker checkbox detection, docker failure logs, run history API, Redis store with startup recovery, SIGHUP rules/log-level reload, Helm OCI publishing, image signing + SBOMs.

**Architecture:** All changes extend existing packages behind existing interfaces. The dispatcher gains a swappable router (atomic pointer) and an optional history recorder; the store gains a Redis implementation selected by config; platform adapters gain repo-scope and marker filters. CI/CD changes are confined to `.github/workflows/release.yml` and `.goreleaser.yaml`.

**Tech Stack:** existing stack + `github.com/redis/go-redis/v9 v9.21.0`, `github.com/alicebob/miniredis/v2 v2.38.0` (tests), `sigstore/cosign-installer v4.1.2`, `anchore/sbom-action v0.24.0`.

```go
// logTail fetches the container's last log lines for diagnostics; the
// container is removed right after, so this is the only chance to see why
// a run failed. Best-effort: errors are reported inline, never fatal.
func (e *Executor) logTail(ctx context.Context, id string) string {
	rc, err := e.api.ContainerLogs(context.WithoutCancel(ctx), id, container.LogsOptions{
		ShowStdout: true, ShowStderr: true, Tail: "50",
	})
	if err != nil {
		return "<failed to fetch logs: " + err.Error() + ">"
	}
	defer rc.Close()
	var buf bytes.Buffer
	if _, err := stdcopy.StdCopy(&buf, &buf, io.LimitReader(rc, 64<<10)); err != nil {
		return "<failed to read logs: " + err.Error() + ">"
	}
	return buf.String()
}
```

```go
type History struct {
	mu      sync.Mutex
	entries []Entry
	idx     int // next write position
	filled  bool
}

func New(size int) *History {
	if size <= 0 {
		size = 100
	}
	return &History{entries: make([]Entry, size)}
}

func (h *History) Record(e Entry) {
	h.mu.Lock()
	defer h.mu.Unlock()
	h.entries[h.idx] = e
	h.idx++
	if h.idx == len(h.entries) {
		h.idx = 0
		h.filled = true
	}
}

func (h *History) Entries() []Entry {
	h.mu.Lock()
	defer h.mu.Unlock()
	n := h.idx
	if h.filled {
		n = len(h.entries)
	}
	out := make([]Entry, 0, n)
	for i := 1; i <= n; i++ {
		out = append(out, h.entries[(h.idx-i+len(h.entries))%len(h.entries)])
	}
	return out
}
```

Validation: `type` ∈ {"", "memory", "redis"}; redis requires `url`. Defaults: KeyPrefix `renovate-server:`, TTL `2h`.

**Design:** one Redis hash per repo, key `<prefix>repo:<repoKey>`, fields `state`, `reason`, `since` (RFC3339Nano), `rerun` ("1"/""). All state transitions via one Lua script each for atomicity; every write refreshes TTL. `Snapshot` = `SCAN <prefix>repo:*` + `HGETALL`. Semantics identical to memory store (Queue/StartRun/FinishRun/Adopt).

Lua for `Queue` (KEYS[1]=key, ARGV: reason, now, ttlSeconds) returning 0=Queued 1=Coalesced 2=Deferred:

```lua
local state = redis.call('HGET', KEYS[1], 'state')
if not state then
  redis.call('HSET', KEYS[1], 'state', 'queued', 'reason', ARGV[1], 'since', ARGV[2], 'rerun', '')
  redis.call('EXPIRE', KEYS[1], ARGV[3])
  return 0
end
redis.call('EXPIRE', KEYS[1], ARGV[3])
if state == 'running' then
  redis.call('HSET', KEYS[1], 'rerun', '1')
  return 2
end
return 1
```

`StartRun`: `HSET state running since <now>` + EXPIRE (only if key exists — `EXISTS` check in script). `FinishRun`: script `local r = HGET rerun; DEL key; return r == '1'`. `Adopt`: script `if EXISTS == 0 then HSET ... running ... end` + EXPIRE.

**Startup recovery (main.go):** after building dispatcher, if store is redis: `for key, st := range st.Snapshot()`: `queued` → parse key (`platform:full/name`, SplitN ":" 2) → `disp.Enqueue(...)` with reason `rerun`; `running` entries whose repo was NOT adopted (dispatcher.Adopt does store.Adopt → key exists; adoption happens first) stay until TTL evicts them. Log both counts.
